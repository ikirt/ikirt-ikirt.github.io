    Ipad 2 Jailbreak

    My iPad is a version 2 from 2012, running 9.3.5. After some reading I found that there is a Jailbreak exploit for 9.3.5(Phoenix), but this is tethered, which means I need to JB after every reboot. More reading finds that I can down grade to 8.4.1 which happens to run faster than 9.3.5 and allows for persistant JB after reboots.

    1. Starting off with a fresh wipe and flash the 9.3.5 firmware to start clean.
    2. Next I had to change the DNS I used as my AdBlock was stopping Cydia from being installed.
    3. Sideload Phoenix.ipa with Sideloady
    4. Run Phoenix to Prepare iPad for JailBreak, running the first time we need to Settings - Device Management , click on your account, and select Trust account-name
    5. Run Phoenix again to perform the Jailbreak.
    6. Go into Cydia and install Filza
    7. Now we modify a file to trick the iPad into thinking it's version 6 instead of 9.3.5 so it will upgrade to 8.4.1
    8. In Filza browse to System > Library > CoreServices
    9. Open up the file named SystemVersion.plist
    10. Update the following, ProductVersion = 6.0 and ProductBuildVersion = 10A403, you can find the correct code here ipsw.me. Search for your device then the firmware version you want.
    11. Save the file and reboot the iPad
    12. Once rebooted, go into Settings > About and check the version on the iPad it should say 6.0
    13. Now Check for an update and downgrade to 8.4.1
    14. Now we need to do a full wipe to create cache files required for 8.3.5 before we JailBreak. Without this Jailbreak will fail at 40% complaining about missing files(cache).
    15. Before installing make sure to remove ADBlocker
    16. Now download etasonJB-RC5 and sideload it with sideloady
    17. After installed open up the app and click "Etason JB!" and wait :)

    NOTES: DFU Mode. To get into this, with iPad powered off, Hold the Home button and power on device or plug into computer, and restore from iTunes.
